sleep

  • sleep(ms: number): Promise<void>
    • Defined in utils/sleep.ts:22

    Sleep for a while. Provides the same functionality as with PHP, Bash etc.

    The function is asynchronous, which means other scripts are able to run while the sleeping script is waiting. As soon as the sleep is over, the script execution will continue.

    example
    async function task () {
 console.log(`Starting task...`)
 console.log(`Going to sleep for 2 seconds...`)
 await v.sleep(1000 * 2)
 console.log(`I'm back!`)
}

    Parameters

    • ms: number

      Number of milliseconds to sleep (1000 ms = 1 s)

    Returns Promise<void>

awaitCondition

  • awaitCondition(condition: (...args: any[]) => any, config?: { args?: any[]; interval?: number; timeout?: number }): Promise<unknown>
    • Defined in utils/await.ts:30

    This function provides extended async/await functionality, by allowing you to await anything.

    example
    let aVariable;
// Set aVariable after 5 seconds
setTimeout(() => {
 aVariable = "Hello";
}, 5000)
// Since the timeout above has to pass before aVariable
// is defined, this console.log will not print any value.
console.log(aVariable)
// => _undefined_
await awaitCondition(() => typeof aVariable !== 'undefined')
// Now it's defined!
console.log(aVariable)
// => "Hello"
